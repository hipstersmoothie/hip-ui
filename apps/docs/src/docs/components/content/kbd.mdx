## Supported Keys

The component supports the following special keys:

- `MetaOrCtrl` - Shows ⌘ on Mac, Ctrl on other platforms
- `Shift` - Shows ⇧
- `Enter` - Shows ↵
- `Tab` - Shows ⇥
- `Backspace` - Shows ⌫
- `Delete` - Shows ⌦
- `ArrowLeft` - Shows ←
- `ArrowRight` - Shows →
- `ArrowUp` - Shows ↑
- `ArrowDown` - Shows ↓
- `Plus` - Shows +
