## Features

### Tokens

The Text component gives you direct acces to all of the [typography tokens](../foundations/typography.mdx).
You can fully customize the `font`, `size`, `weight`, and `tracking` properties.

> NOTE: By using this component you are not using the [typeramp](./typography.mdx) component.
> It could lead to inconsistent typography across your application.
